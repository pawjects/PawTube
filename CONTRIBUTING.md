# Contributing to PawTube

First off, thank you for considering contributing to PawTube! It's people like you that make the open-source community such a great place to learn, inspire, and create.

## 🛠️ Development Setup

1. Fork the repo and create your branch from `main`.
2. Clone your fork: `git clone https://github.com/your-username/PawTube.git`
3. Since this is a static web utility, you can simply open `index.html` in your browser or run a local live server to see your changes.

## 🎨 UI/UX Guidelines

If you are contributing to the frontend, please adhere to our design principles:
* **Theme:** Material Design 3 (MD3) Dark Monet.
* **Colors:** Rely on pure AMOLED black backgrounds (`#000000`) and subtle cyan gradients. Avoid bright neon colors.
* **Icons:** strictly use standard web icons (Material Symbols). Do not use emojis or external SVGs for UI elements.
* **Layout:** Ensure all interfaces remain distraction-free and fully mobile-responsive.

## 📝 Pull Request Process

1. Ensure your code follows the existing style and design guidelines.
2. Update the README.md with details of changes to the interface, if applicable.
3. Submit your PR and request a review from the maintainers.
